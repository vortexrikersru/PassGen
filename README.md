Secure Password Generator
A modern, responsive, and user‑friendly web application for generating strong, customizable passwords. Built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies, just clean and efficient code.

Features
• 	Customizable Password Options
• 	Uppercase letters
• 	Lowercase letters
• 	Numbers
• 	Symbols
• 	Adjustable Length Slider
Choose any length from 4 to 32 characters with real‑time updates.
• 	Dynamic Strength Meter
Visual indicator that evaluates password strength based on:
• 	Length
• 	Character variety
• 	One‑Click Copy to Clipboard
Includes a smooth toast notification for confirmation.
• 	Auto‑Generate on Page Load
A secure password is created instantly when the app opens.
• 	Modern UI & Responsive Design
Clean layout, gradient background, subtle animations, and mobile‑friendly components.
